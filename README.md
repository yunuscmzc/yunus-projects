# Time Domain and Frequency Domain Analysis of the Coffee Dataset

This project analyzes the Coffee dataset using both time-domain and frequency-domain techniques to evaluate their usefulness for classification. The study includes descriptive statistics, representative time-series visualization, autocorrelation analysis, and spectral analysis via periodograms and dominant frequency extraction. Robustness is assessed through controlled noise injection and its effect on spectral detectability. A feature-based classification approach using dominant frequency, spectral concentration, peak power, and short-lag autocorrelation is implemented with a nearest-centroid classifier. Results show that while spectral features are stable under noise, they provide limited discrimination compared to benchmark methods, highlighting the importance of richer temporal representations.
